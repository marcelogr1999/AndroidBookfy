# AndroidBookfy
Aplicação em Android de 'Bookmarking' de livros.

FUNCIONALIDADES:

.Geral
- Login
- Visualizar lista geral
- Adicionar livros a sua lista pessoal
- Escolher status para esses livros
- Alterar status
- Remover livros de sua Lista
- Alterar senha
- Criar novo usuário
- Compartilhar lista
- Notificações
